Features

Royal Testimonial version 1.0 has come with the following feature. However, we've planned to extend it a lot that find our customers very helpful to create a full featured testimonial manager.

  • Total testimonials, pending testimonials, approved testimonials and total forms report section.
  • Fully drag and drop testimonial form builder.
  • Come with 5 default form style.
  • Come with basic, grid and carousel testimonial style.
  • Built in iFrame code generator with dynamic width and height field.
  • Can be also integrated with your website without using iFrame. However, you may need to have the little bit of PHP and CSS skills.
  • Image field enables or disables option.
  • Email field enable or disable option
  • Position field enable or disable option
  • Company field enable or disable option
  • Website field enable or disable option
  • New testimonial submission email notification.
  • Testimonial approve, disapprove or decline option.
  • Administrator profile update feature.

Installation

Royal testimonial has been developed by using raw PHP. So the installation process would be very easy on any server. However, here's the relevant installation process which I hope you find helpful:

  • Extract the royal-testimonial.zip file on your computer that you've downloaded from CodeCanyon.
  • Go to extracted folder.
  • Upload royal.zip file to your server by using any FTP application (e.g. FileZilla).
  • Please, set the royal/admin/class/config and royal/admin/asset (Including all the subfolder) folder permission to read and write mode (775). You can set the folder permission by using either shell commands or using your hosting cPanel file manager change permission option. Here's the relevant screenshot which I hope you find helpful:

  • After you set the proper folder permission, please create a new database and make a note of the database name, database username, database password and the hostname.
  • After creating a new database successfully please go to the admin area to install the application (e.g. http://yourdomain/admin/).
  • The application will redirect you to the installation wizard page.
  • Please, write the hostname (e.g. localhost), database name, database username and database password in the installation wizard fields to run the installation process.

  • If all the database credentials are correct, the application will redirect you to the admin page.
  • The application will install a default account to log in to the admin section for the first time. The default username would be: admin and password would be: admin.
  • You can also login to the Royal testimonial by using this URL: http://yourdomain/admin
  • Once you're logged into the admin area please change your username, password and other information from the profile page.

Settings

Settings Page

Select a form style

On the Select a form section, you will find an option field that contain all the default forms. We already created five forms style to help you. To view the forms how those look like just select the form that you want to see and click view button to view it. If you decide to set that form then click the change button. You can also delete forms by clicking the delete button. If you want to show that form on your site via iframe then just click get the code and it will generate code for your site, copy and paste that whenever you want to show. You can also customize your iFrame dimension.

Here's the relevant screenshot which I hope you find helpful:

You can also view the default form or add the default testimonial submission form link to your website by using this URL: http://yourdomain/submit.php

Select a testimonial style

From the select a testimonial style section you can pick a style and save it. If you want to see the style then you can click the view button. If you want to show your testimonials on your site via iFrame then click the get code button and copy the relevant iFrame code and place the relevant code it into your site where you wish to display the testimonial. You can also delete a specific testimonial style if you like.

Here's the relevant screenshot which I hope you find helpful:

You can view the default testimonial style or add a link to your website by using this URL: http://yourdomain/index.php

Image Field

From the image field switch, you can either turn off or turn on the image field on the testimonial style area.

Email Field

From the email field switch, you can either turn off or turn on the email field on the testimonial style area.

Position Field

From the position field switch, you can either turn off or turn on the position field on the testimonial style area.

Company Field

From the company field switch, you can either turn off or turn on the company field on the testimonial style area.

Website Field

From the website field switch, you can either turn off or turn on the website field on the testimonial style area.

MailGun Service

From the MailGun field switch, you can either turn off or turn on the email notification when a new testimonial is submitted. You'll also need to have a MailGun account and add your MailGun domain name and secret API key from the MailGun settings fields.

Getting a MailGun API Key as simple as just create an account on mailgun.com. We've implemented Mailgun API Key to sending the email to ensure all the email hit the users inbox, not to the spam folder.

  • Please, create an account on mailgun.com. It’s free and doesn’t require any credit card.
  • You’ll now find your API credentials in your mailgun.com account dashboard.
  • Please, make a note of the API Key credentials to add these with the relevant settings.

 

Usage

Dashboard

On the dashboard, you will find four reports sections where every section allows you to access shortly and overview of forms and testimonials. Four section you will find:

Total Forms

From the total forms report section, you can see how many form you have been created or exist. By clicking view all forms, you will see settings page where all exist form will be shown.

Total Testimonials

From the total testimonials report section, you can see how many testimonials you have been received or exist. By clicking view all testimonials, you will see testimonials page where all the exist pending, approved testimonials.

Pending Testimonials

From the pending testimonials report section, you can see how many testimonials you have pending. By clicking view pending testimonials, you will see the testimonials page where all the exist pending, approved testimonials.

Approved Testimonials

From the total approved testimonials report section, you can see how many testimonials you have approved yet. By clicking view approved testimonials, you will see testimonials page where all the exist pending, approved testimonials.

Builder

From the builder page, you can make your own customized testimonial form by using the form builder for testimonial submission. The builder has two parts. On the middle is the editor and fields are on the right side. You can make the form  by just simply dragging and dropping the field on the editor area. Before you make a form you have to keep in mind something important about the fields. To keep the royal testimonial specific design you have to use some of the default names on the form fields.

Here is the default field list for some of the particular form fields:

  • If you want to create a name field using the text field then please add the "name" by editing the text field. You can also use any other field instead of using the text field.

Here's the relevant screenshot which I hope you find helpful:

  • If you want to create a position field using then please add a text field and then please add the "position" from the name label by editing the text field. This above screenshot is true for all these relevant fields.
  • To add a company name specific style on the testimonial please add the "companyname" on the form field name option.
  • To add a company website specific style on the testimonial please add the "companywebsite" on the form field name option.
  • To add an email field specific style on the testimonial please add the "email" on the form field name option.
  • To add a feedback field specific style on the testimonial please add the "feedback" on the form field name option.
  • To add the rating specific style on the testimonial please add the "rating" on the form field name option.
  • To add an image specific style on the testimonial please add the "image" on the form field name option.

Why have defaulted Names?

The Royal testimonial come with 3 testimonial style as such if you want to keep your style you'll need to use the pre-defined name or the still will be different.

How to Add an Image Field?

Royal testimonial doesn’t allow customers to upload their photo but doesn't worry, we've covered you. You can simply make a text field and add the "image" on the name field. Your customers now can enter their direct image link into the relevant field and the Royal testimonial will take care of the rest. All the images will appear correctly as expected.

Customization

This app has come with pre-defined styles which is fully customizable. If you'd like to customize the default styles then please customize the index.php, feedback-basic.php, feedback-grid.php, feedback-carousel.php or submit.php, submit-basic.php, submit-grid.php and submit-carousel.php file from the royal/ directory. You can add your custom stylesheet from the royal/admin/asset/css directory and js/ for the custom Javascript.
